# yuketang-mark-helper
雨课堂作业批改助手

[从 GitHub 安装](https://raw.githubusercontent.com/hotwords123/yuketang-mark-helper/gh-pages/yuketang-mark-helper.user.js)

## 功能

1. 将学生提交的图片附件转换为图片内容，便于直接查看和批注。
2. 为图片和 PDF 的批注界面添加了快捷键：
   - `1`, `2`, `3`：切换批注模式（选择、画笔、文本）。
   - `Q`, `W`, `E`：切换画笔和文本尺寸（仅支持图片批注）。
   - `Ctrl + Z`：撤销上一步操作
   - `Ctrl + Y`：重做上一步操作（仅支持 PDF 批注）
   - `Esc`：不保存批注并退出图片批注界面
   - `Enter`：保存批注并退出图片批注界面
3. 修复了 PDF 批注界面使用手写笔时页面会滚动而无法正常书写的问题。
